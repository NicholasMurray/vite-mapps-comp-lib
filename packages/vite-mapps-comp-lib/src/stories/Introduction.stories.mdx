# Welcome to Version 1 MAPPS Component Library

## Installation

```
npm install mapps-component-library
```

## Importing components

Import the components into your project using the named imports convention. For example:

```
import { Alert, Button, Panel } from 'mapps-component-library';
```

## Styles, Fonts and Images

Because MAPPS Component Library uses .GOV.UK, HO and MOJ styling and class names, 
we don't ship with any included CSS. However stylesheets are required to use 
these components.

### SCSS
The easiest way to include component styles to your application

#### Install GOV.UK, HOds and MoJ npm packages as required
``` 
npm install govuk-frontend --save
npm install @hods/alert --save
npm install @ministryofjustice/frontend --save
```

#### Import GOV.UK, HOds and MoJ SCSS files as required
```
@import "node_modules/govuk-frontend/govuk/all";
@import "node_modules/@hods/alert/assets/Alert";
@import "node_modules/@ministryofjustice/frontend/moj/all";
```

### Fonts and Images

Copy the contents of relevent assets folders to your apps assets folder
```
/node_modules/govuk-frontend/govuk/assets/images folder to <YOUR-APP>/assets/images
/node_modules/govuk-frontend/govuk/assets/fonts folder to <YOUR-APP>/assets/fonts

/node_modules/@ministryofjustice/frontend/moj/assets/images folder to <YOUR-APP>/assets/images
/node_modules/@ministryofjustice/frontend/moj/assets/fonts folder to <YOUR-APP>/assets/fonts
```
